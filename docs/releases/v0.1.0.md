# Weavatrix Online 0.1.0

0.1.0 introduces the separately versioned, source-available Online overlay for
the MIT Weavatrix 0.3 core.

## Product boundary

- The package depends on `weavatrix ^0.3.0`; it does not copy or fork core graph,
  parser, LSP, Health or baseline analyzer implementations.
- It composes the core in-process and may add proprietary MCP tools, skills and
  supplemental local analyzers through the public extension API.
- All HTTP, authentication, sync, advisory refresh and hosted architecture pull
  behavior lives in this package. The MIT core remains network-free.

## Online surface

- `online_status` negotiates the formal Cloud/Enterprise capability document and
  reports compatibility before any upload.
- `refresh_advisories` refreshes the explicit local advisory cache.
- `preview_sync` creates the bounded source-free payload and confirmation hash
  without networking.
- `sync_graph` sends only that confirmed payload to the configured HTTPS target.
- `pull_architecture_contract` downloads an owner-approved architecture policy.

The Online profile exposes 39 tools: all 34 core tools plus these five explicit
network-capable extensions.
